# 🚀 QuantX - Smart Trading Landing Page

## 📌 Project Overview

QuantX is a modern, responsive landing page designed for a smart trading platform. It showcases features like backtesting, analytics, and live trading insights with a clean and engaging UI.

This project was built as part of a technical task to demonstrate frontend development skills, UI/UX design, and responsiveness.

---

## ✨ Features

* 📱 Fully Responsive Design (Mobile + Desktop)
* 🧭 Navigation Bar with smooth scrolling
* 🎯 Hero Section with Call-To-Action
* ⚡ Features Section (Backtesting, Analytics, Signals)
* ℹ️ About Section
* 🚀 Call-To-Action Section
* 📌 Footer with project info
* 🎨 Modern UI with hover effects
* 🔄 Smooth scrolling using JavaScript

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript
